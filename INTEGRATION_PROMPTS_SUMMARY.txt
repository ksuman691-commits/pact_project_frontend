HOW TO COPY-PASTE INTO YOUR LLM
====================================================================

STEP 1: Choose your LLM
- Claude (Cursor/VS Code)
- ChatGPT (web)
- Deepseek
- Gemini
- Any other LLM

STEP 2: Open the prompt file
If using QUICK_START_PROMPT.txt:
  1. Open in your editor
  2. Read issue by issue
  3. Copy code snippet
  4. Paste into your LLM with: "Fix this in my code:"

If using COPY_PASTE_PROMPT.txt or VSCODE_MASTER_PROMPT.md:
  1. Open file in your editor
  2. Select all (Ctrl+A)
  3. Copy (Ctrl+C)
  4. Open your LLM chat
  5. Paste (Ctrl+V)
  6. Add this at the end: "Help me implement these fixes step by step"

STEP 3: Work with LLM
  - Ask for Issue #1 first
  - Ask which files to modify
  - Ask for code examples
  - Paste LLM's response into your editor
  - Test with Network tab
  - Move to next issue

STEP 4: Commit after each issue
  git add -A
  git commit -m "Fix Issue #X: [description]"
  git push origin your-branch-name
